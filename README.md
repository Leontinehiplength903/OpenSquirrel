## 🧭 What OpenSquirrel Does

OpenSquirrel is a Windows app for people who use more than one coding agent at once. It gives you one place to run Claude Code, Codex, Cursor, and OpenCode side by side.

Use it when you want to compare results, switch fast, and keep your work in one window. It runs as a native Rust app with a clean interface built on GPUI.

## 💻 What You Need

OpenSquirrel is built for Windows.

You will need:

- Windows 10 or Windows 11
- A mouse and keyboard
- Internet access for setup and agent sign-in
- Enough free disk space for the app and its local data

For best results, use:

- A screen size of 1366 x 768 or higher
- 8 GB of memory or more
- A recent CPU from Intel or AMD

OpenSquirrel keeps your work on your machine unless a connected agent service needs internet access.

You may be asked to sign in to each tool you use. That sign-in happens with the service itself. OpenSquirrel acts as the control plane that brings the tools into one view.

Your session data, layout choice, and local settings may stay on your PC so the app can reopen the same workspace later.

## 🔍 Troubleshooting

### The app does not open

- Check that the download finished
- Make sure you extracted the zip file if needed
- Try right-clicking the app and choosing **Run as administrator**
- Restart your PC and try again

### Windows blocks the file

- Open the file properties
- Check whether Windows marked it as blocked
- Unblock it if the option appears
- Run the app again

### A panel stays blank

- Refresh the panel
- Check your internet connection
- Sign in again if the agent service signed you out
- Close and reopen the app

### The app looks too small

- Change your Windows display scale
- Use a higher screen resolution if possible
- Increase the font size in the app settings
